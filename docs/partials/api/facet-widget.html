facet-widget

Description

A UI control for a SOLR facet. Displays the available content as a set of checkboxes that can be added to the query as filters.

Usage

as element:
<facet-widget
       facet-field="{string}"
       label="{string}">
</facet-widget>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
facetFieldstring
  • The field for which to get the facet counts
labelstring
  • The name to be used for the widget
