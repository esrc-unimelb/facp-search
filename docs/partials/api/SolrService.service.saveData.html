saveData
function in module SolrService

Description

Pass it a SOLR response and it manages the data object used by the interface.

This method knows how to handle no result found as well as new data via infinite scroll.

The message 'search-results-updated' is broadcast via $rootScope when the data is ready to go. Any widget that interacts with the data should listen for this message.

Usage

saveData(d);

Parameters

ParamTypeDetails
dobject
  • The SOLR response
